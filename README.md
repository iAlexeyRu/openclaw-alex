# openclaw-alex

Живой workspace Алисы для Алекса.

Что здесь хранится:
- identity и persona-файлы (`IDENTITY.md`, `SOUL.md`)
- профиль Алекса (`USER.md`)
- правила и устройство workspace (`AGENTS.md`)
- локальные заметки по инструментам (`TOOLS.md`)
- heartbeat-настройки (`HEARTBEAT.md`)
- память и рабочие записи (`memory/`, `MEMORY.md` по мере появления)
- другие полезные файлы, которые Алиса создаёт по ходу работы

## Принцип

Это не только стартовый шаблон. Это рабочий репозиторий, в котором Алиса хранит свой актуальный workspace и ведёт его вместе с Алексом.

## Что обычно не коммитится

- `.openclaw/` — локальное служебное состояние рантайма
- приватные секреты и токены
- временный мусор и кэш
